Bu eklenti, Intense kargo entegrasyonları veya manuel kargo eklentisini geçmişte kullanıp, artık kullanmayan kullanıcılarımız için geliştirilmiştir.

== Description ==
DİKKAT: Bu eklenti bir kurtarma eklentisidir, bu eklenti sadece geçmişte Intense Yazılım kargo eklentilerini kullanmakta olup "artık kullanmayan" kullanıcılar içindir.

CAUTION: the plugin for only users that was used intense.com.tr shipping plugins in the past but not using currently.

Intense Kargo eklentileri kullanıcılarının eklentiyi kullanmayı bıraktıklarında "Kargoya verildi" sipariş durumunun sistemde olmaması sebebiyle görünmeyen siparişleri görünür hale getirir.

Özellikler:
* "Kargoya verildi" durumunu aktif hale getirir.
* Siparişe ilişkin kargo firması ve kargo takip no verilerini yönetici sipariş detayından görebilmesinizi sağlar (düzenleme özelliği bulunmamaktadır.)

Bu eklenti intense.com.tr tarafından geliştirilmiştir.

== Installation ==
Eklentiyi yükledikten sonra, ayarlar bölümüne gidiniz(Wordpress admin sol menü -> ayarlar -> Intense Kargo Kurtarma Ayarları) ve ilgili ayarları tamamlayınız.
